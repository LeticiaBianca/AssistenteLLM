# AS05: Assistente Conversacional Baseado em LLM

## Como rodar localmente
1. Baixe os arquivos;
2. Instale dependências:
   pip install -r requirements.txt
4. Rode usando:
   streamlit run app.py

## Como funciona
- Faz upload de PDF;
- Indexa o texto com embeddings;
- Responde perguntas;
